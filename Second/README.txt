Данный проект содержит два приложения, оба имитируют работу музыкального плеера, но
имеют разные способы внедрения зависимостей

Первое сконфигурировано при помощи XML файла, все зависимости для бинов поставлятся через него

Второе скофигурировано при помощи классов-аннотаций, все зависимости поставляются через аннотации из текстового
файла
